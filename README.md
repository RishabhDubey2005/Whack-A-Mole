# Whack-A-Mole
Whack-A-Mole is a simple game that is intended to incorporate the Python Pygame Package

Objective:
    - A user is given 1 minute to "whack" as many moles as possible
    - The user is awarded 1 point for every correct mole they hit.
    - The user loses 1 point for every incorrect mole they hit.
    - Once the timer is done, the game will show the user their final score.


Template:
Home Screen:
    - Will show users with either the play button or the rules button
    - Exit button should also be provided such that the program will exit without having to provide an interruption within the Terminal

Rules Screen:
    - Shown if the user clicks the rules button from the home page
    - Should explain the basic rules of the game
    - Back Button should be incorporated to allow the user to go back to the home screen

Play Screen:
    - Where the actual gameplay takes place

Result Screen:
    - Activated once the timer goes off in the Play Screen
    - Should Provide a button to either play again or exit the application
